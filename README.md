Explorador interactivo del multiverso de **Rick and Morty** que consume la API pública [rickandmortyapi.com](https://rickandmortyapi.com).

**"¡Wubba Lubba Dub Dub!"**

## ✨ Características principales

- Diseño cyber-futurista con gradientes y glassmorphism
- Búsqueda en tiempo real de personajes por nombre
- Visualización de **Personajes**, **Episodios** y **Locaciones**
- Tarjetas dinámicas generadas con JavaScript (template literals)
- Manejo completo de estados: **Cargando**, **Éxito** y **Error**
- Totalmente responsive (mobile-first)
- Soporte para tecla **Enter** en el buscador

- **Tailwind CSS** (vía CDN)
- Fetch API para consumir datos
- Google Fonts + Material Symbols
